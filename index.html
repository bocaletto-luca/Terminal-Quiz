Terminal Quiz Multiplayer

A lightweight, real-time multiplayer quiz game for Linux terminals.

View on GitHub

Features

  • Real-time multiplayer quiz over TCP (minimum 2 players).
  • Secure registration & login with SHA-256 password hashing.
  • JSON-driven question database (`questions.json`).
  • Session scoring: +10 points correct, –5 points wrong or timeout.
  • Persistent global leaderboard (`data.json`) and in-game `leaderboard` command.
  • Zero external dependencies—runs on Python 3.6+ with standard library only.

Installation

  1. Clone the repository: 
    git clone https://github.com/bocaletto-luca/terminal-quiz-multiplayer.git
cd terminal-quiz-multiplayer
  2. Optionally make scripts executable: 
    chmod +x server.py client.py
  3. Ensure Python 3.6+ is installed on your Linux system.

Usage

Start the Server

./server.py

The server listens on TCP port 5000 and automatically generates data.json.

Launch Clients

./client.py

In each terminal session, connect and use these commands:

  • register – Create a new user.
  • login – Log in with existing credentials.
  • start – Queue for a quiz session.
  • leaderboard – View the global top 10.
  • exit – Disconnect from the server.

Customizing Questions

Edit questions.json to add or modify quiz items. Each entry must follow:

{
  "id":       6,
  "question": "Your question text?",
  "options":  ["Option A", "Option B", "Option C", "Option D"],
  "answer":   2        // zero-based index of correct option
}
